## INSTALL

- Download respository
- Create the database with the name vetegram
- To load libraries execute: composer install
- To create or update database execute: php artisan migrate

## UPDATE DATABASE

- execute: php artisan migrate

## REQUIREMENTS

- PHP 7.3 (or greater)
- MySQL 5.0.15 (or greater)
